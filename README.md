# The MXNet Implementation of ShuffleNet v1 and v2

## Environment

-   Python 2.7 
-   Ubuntu 18.04
-   Mxnet-cu90 (=1.3.0)

## Usage

To test ShuffleNet v1 and v2 on mnist
```
python test_mnist.py
```
